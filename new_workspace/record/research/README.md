# Research Workspace

This folder is organized into three top-level parts:

- `docs/`
  Research methodology, experiment design, and generated narrative reports.
- `code/`
  Research-only implementation for subgroup construction, personalization experiments, and report generation.
- `results/`
  CSV, Markdown, and manifest outputs generated by the research runners.

## Main Code Entry Points

- `code/experiment_runner.py`
  Builds subgroup tables and runs baseline / personalization experiments.
- `code/report_builder.py`
  Builds the consolidated baseline-and-personalization report from generated results.

## Main Commands

```bash
python -u /Users/xujing/FYP/new_workspace/record/research/code/experiment_runner.py --mode build-subgroups
python -u /Users/xujing/FYP/new_workspace/record/research/code/experiment_runner.py --mode subgroup-baseline
python -u /Users/xujing/FYP/new_workspace/record/research/code/experiment_runner.py --mode personalize-l1
python -u /Users/xujing/FYP/new_workspace/record/research/code/experiment_runner.py --mode personalize-l2
python -u /Users/xujing/FYP/new_workspace/record/research/code/experiment_runner.py --mode personalize-l3
python -u /Users/xujing/FYP/new_workspace/record/research/code/report_builder.py
```
